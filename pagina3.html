CSS é um documento de estilização em cascata criado em 1993, com sua primeira versão amplamente utilizada em 1997. A linguagem CSS permite definir regras que definem características para os elementos HTML como cor, borda e comportamentos, além de animações e transições.


Ao longo dos anos, o CSS evoluiu e hoje temos o CSS 1, 2 e 3, cada uma das versões adicionou novos recursos que eram demandados no período

  1. CSS 1 1997 - Propriedades de estilização e formatação básicas, além de controle do posicionamento dos elementos html com propriedades float e position
  2. CSS 2 2000 - Adição de propriedades de animação, transição e melhoria de position
  3. CSS 3 2008 - Incremento gradual de novas funcionalidades do CSS com animação e ferramentas de posicionamento com display: flex(2008) e grid(2013)
