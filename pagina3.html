O CSS, documento de estilização em cascata criado em 1993, com sua primeira versão amplamente utilizada em 1997. A linguagem CSS permite definir regras que definem características para os elementos HTML como cor, borda e comportamentos além de animações e transições.

Ao longo dos anos o CSS evoluiu e hoje temos o CSS 1, 2 e 3, cada uma das versões adicionou novos recursos que eram demandados no período.

  1. CSS 1 1996 - propriedades básicas de estilização e formatação, além de controle do posicionamento dos elementos HTML com propriedades float e position.
  2. CSS 2 1998 - adição de propriedades de animação e transição e nelhoria do position.
  3. CSS 3 2008 - incremento gradual de novas funcionalidades do CSS com animação e ferramentas de posicionamento como display:flex (2008) e grid (2013).

Uma regra CSS pode ser definida com a seguinte sintaxe 

            Seletor{
                Propriedade 1:Valor;
                Propriedade 2:Valor;
                Propriedade 3: Valor;
            }

Podemos incluir código CSS de três formas diferentes

Formas de incluir CSS
CSS externo
Quando definimos as regras CSS a serem utilizadas por meio de um arquivo externo .css, vinculando ele ao HTML por meio da tag <link >
CSS incorporado
Quando definimos código CSS pela tag <style >
CSS inline
Quando definimos código CSS de estilização pelo próprio elemento HTML como: Texto de cor vermelha com CSS inline
